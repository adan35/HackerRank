# Program to find the Nth Prime Number

Given an integer __N__. The task is to find the __Nth__ prime number.

#### Examples:  

<pre>
  <div style = "background: #F9F9F9">
    <b>Input:</b> N = 5
    <b>Output:</b> N = 11

    <b>Input:</b> N = 16
    <b>Output:</b> N = 53

    <b>Input:</b> N = 1049
    <b>Output:</b> N = 8377
  </div>
</pre>
